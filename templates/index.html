ptpb


ptpb is a simple pastebin.


USE


GET:

Returns an existing paste.

    curl https://ptpb.pw/AAAA

POST:

Creates a new paste, returns URL and UUID.

- 'multipart/form-data' (preferred)

    curl -F c=@- https://ptpb.pw

- 'application/x-www-form-urlencoded' (less-preferred)

    curl -d @- https://ptpb.pw

PUT:

Modifies an existing paste. Requires the paste UUID for authentication. 

    curl -X PUT -F 'c=@-' https://ptpb.pw/529441ee-0d09-4465-81bc-cf0d2cfb1d18

DELETE:

Deletes an existing paste.

    curl -X DELETE https://ptpb.pw/529441ee-0d09-4465-81bc-cf0d2cfb1d18


ENDPOINTS


/ - Displays this page, handles all post/get/delete requests.
/<ID> - View a paste.
/<ID>/<LEXER> - Display a paste with the syntax highlighting of the lexer you select. 
/f - Webform to submit a paste.
/s - View statistics. 


OTHER INFO


Make an function for more convenient access:
    ptpb(){ curl -F "c=@$1" https://ptpb.pw }

Examples:
    ptpb cats.jpg

    dmesg | ptpb -

SSL is now enabled and forced.
